# Papier
Mobile online shop application (no third party included, except `navigation-compose`)
> `API:` https://eshop.jemaristudio.id/api

## Features
- Basic Authentication
- View & Search `Product`
- Add `Variant` (of a product) to `Cart`
- Remove `Variant` (of a product) from `Cart`
- Change the quantity of an item in `Cart`
- Checkout all items in `Cart` (with optional coupon code)
- View `Transaction` history
